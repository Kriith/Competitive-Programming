# Competitive Programming

The goal of this repository is to serve as a collection of my solutions, algorithms, and approaches to help with learning and practicing problem-solving skills.

## Repository Structure

```text
.
├── [Codeforces](./Codeforces/)
├── [LeetCode](./LeetCode/)
├── [CSES](./CSES/)
├── [AtCoder](./AtCoder/)
├── ...
├── [notes.md](./notes.md)
├── [revision.txt](./revision.txt)
└── [macro.cpp](./macro.cpp)
```

* Platform folders contain solutions to individual problems.
* Codeforces contest folders are named using the contest ID (e.g. `2074/`).
* LeetCode solutions are named using the problem's slug (e.g. `two-sum.cpp`).
* [`notes.md`](./notes.md) contains useful algorithms, observations, and implementation tricks.
* [`revision.txt`](./revision.txt) lists problems worth revisiting.
* [`macro.cpp`](./macro.cpp) contains the boilerplate template that I use during Codeforces contests.
